## ❓ 常见问题

### 如何快速开始？
1. 激活虚拟环境：`.\AI_Server\Scripts\Activate.ps1`
2. 运行项目：`python run.py`

### 如何检查环境？
运行 `python scripts/setup.py` 检查环境状态。

### 如何启动项目？
运行 `python run.py` 启动主程序。
